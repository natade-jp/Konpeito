    public constructor(number: BigInteger) source

    Create an arbitrary-precision integer.

    Initialization can be performed as follows.

    • 1200, "1200", "12e2", "1.2e3", ["1200", 10]
    • "0xff", ["ff", 16]
    • "0o01234567", ["01234567", 8]
    • "0b0110101", ["0110101", 2]

    Params:
