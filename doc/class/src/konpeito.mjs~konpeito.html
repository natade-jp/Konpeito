    konpeito

    計算に利用できるデータを提供するクラス 大まかに、 BigInteger, BigDecimal, Matrix の3つに分かれる。 Matrix は、 Complex を包括している。 多倍長整数演算を特化した計算クラスは、 BigInteger 。 任意精度浮動小数点演算を特化した計算クラスは、 BigDecimal 。 信号処理や統計処理等を備えた汎用的な計算クラスは、 Matrix 。
